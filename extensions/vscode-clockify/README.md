# VS Code Clockify Extension

Start and stop Clockify timers based on GitHub repo context with interactive selection.

## Setup
- Install the extension from this folder.
- Run "Clockify: Set API Key".

## Usage
- Run "Clockify: Start Timer for Issue" to pick workspace, project, task, and tags.
- The extension remembers your last selection per repo.
- Use the status bar item to start/stop quickly.
