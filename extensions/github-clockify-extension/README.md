# GitHub Clockify Starter Extension

Chrome extension that adds a "Start Clockify" button to GitHub issue and PR pages.

## Setup
- Load the extension from this folder in Chrome.
- Open extension settings and enter your Clockify API key and workspace id.
- Add repo mappings and label tag mappings as needed.

## Auto-match
- Repo mappings: map `owner/name` to project/task ids.
- Label to tag map: map issue labels to Clockify tag ids.
